# Verilog Digital Design Lab

This repository contains Verilog HDL programs developed as part of digital design laboratory practice. It is organized in a structured way to cover fundamental concepts of digital electronics, starting from basic arithmetic and combinational circuits to sequential circuits, finite state machines, and application-based designs.

The repository is intended for learning, practice, and academic reference. Each folder contains Verilog modules, testbenches, and short explanations for the corresponding experiments.

## Folder Structure

- **01_Arithmetic_Circuits**  
  Includes basic arithmetic circuits such as half adder, full adder, subtractors, and multipliers.

- **02_Combinational_Building_Blocks**  
  Contains combinational logic designs like code converters, multiplexers, demultiplexers, decoders, and encoders.

- **03_Sequential_Circuits**  
  Covers sequential logic circuits such as flip-flops, shift registers, and counters.

- **04_FSM_Designs**  
  Includes finite state machine based designs such as sequence detectors, serial adders, and parity detectors.

- **05_Application_Based_Designs**  
  Contains practical Verilog designs such as traffic light controller and vending machine controller.

- **06_Term_Project**  
  Reserved for a complete project that combines multiple Verilog design concepts.

## Purpose of this Repository

- To practice Verilog HDL systematically
- To understand digital design concepts through coding
- To maintain lab experiments in an organized format
- To build a strong foundation for future FPGA and hardware design projects

## Tools Used

- Verilog HDL
- Xilinx / Vivado / simulation tools
- Testbench-based verification
